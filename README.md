# Study-OS
# 📚 Study OS

A lightweight, browser-based **study operating system** designed to boost productivity with built-in tools like a Pomodoro timer, task manager, notes, and more — all inside a clean desktop-style interface.

---

## 🚀 Features

### 🔒 Lock Screen

* Displays **current time**
* Shows **battery percentage**
* Motivational **daily quotes**
* Smooth unlock animation

---

### 🖥️ Desktop Interface

* Floating **window-based apps**
* **Draggable windows** with saved positions
* Stylish **dock UI** with hover animations

---

### ⏱️ Pomodoro Timer

* Custom **study & break durations**
* Start, pause, and reset controls
* Automatic switching between study/break modes

---

### ✅ Task Manager

* Add and remove tasks
* Tasks saved using **localStorage**
* Click to delete completed tasks

---

### 📝 Notes App

* Simple text editor
* Auto-saves notes in browser

---

### 📊 Stats (Expandable)

* Placeholder for productivity tracking
* Can be extended to show study analytics

---

### 🧮 Calculator

* Quick access advanced calculator.

---

### 🌐 Web Access

* Open:

  * Google (Chrome shortcut)
  * ChatGPT
  * Spotify
  * Notion(NEW)
  * NotebookLM(NEW)

---

### 🎨 Customization

* Change wallpaper:

  * Default options(2 NEW ADDED!!!)
  * Custom image URL
* Saved preferences persist across sessions

---

## 💾 Data Storage

All data is stored locally using:

* `localStorage`

This includes:

* Tasks
* Notes
* Timer settings
* Window positions
* Wallpaper

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **Vanilla JavaScript**

No external frameworks or libraries required.

---

## 📌 Usage Tips

* Click anywhere on the lock screen to unlock
* Drag windows by the title bar
* Click icons in the dock to open apps
* Your data will automatically save in your browser

---

⚠️ Limitations
Works only in modern browsers
No cloud sync (local device only)
Some features (like battery API) may not work on all devices
